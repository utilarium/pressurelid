## [1.0.0] - 2026-01-13

### Added

- Initial release
- `SafeRegex` class with pattern analysis
- `create()` method for safe regex creation
- `testWithTimeout()` for timeout-protected execution
- `globToRegex()` for glob pattern conversion
- `fromUserInput()` for escaped literal matching
- Convenience functions: `createSafeRegex()`, `globToSafeRegex()`, `escapeForRegex()`
- Configurable limits and callbacks
- Comprehensive test suite with ReDoS vectors
- Full documentation
